The scripting calculator using the text file.
 
File contains next format of data: 
         one plus three-two multiply one-one 
         eight-one divide-by curly-brace-open three multiply four minus three curly-brace-close
Result of each operation is appended to the end of the string. Sample result:
         one plus three-two multiply one-one = three-seven-four
         eight-one divide-by curly-brace-open three multiply four minus three curly-brace-close = nine

This sample shows how to work with:strings,files,nested functions.
